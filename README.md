# ATM Simulation(ENG)

This is a simple ATM simulation project developed in Python.  
It allows basic banking operations through a command-line interface and supports both English and Turkish languages.

## Features

- PIN-based user login  
- Balance inquiry  
- Deposit money  
- Withdraw money  
- Transaction history  
- Multilingual support (English & Turkish)

## How to Run

1. Make sure Python 3 is installed on your system.  
2. Open a terminal and navigate to the project folder.  
3. Run the following command:

```bash
pyton atm_simulation.py
